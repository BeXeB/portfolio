# XLSX Converter

## About

A lightweight web tool for converting a specific legacy spreadsheet format into a platform-compatible structure via simple drag-and-drop.

Built for teachers to eliminate repetitive manual data entry when transferring records into a digital system.

---

## Features

- One-step conversion: Upload a file and receive a correctly formatted output instantly
- Fixed schema mapping: Transforms a known input format into the required target structure
- Client-side processing: Fast, in-browser conversion using SheetJS

---

## Tech Stack

- Angular
- TypeScript
- SheetJS
- CSS/SCSS
